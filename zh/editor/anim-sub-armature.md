# 子骨架动画

## 自动关键帧
1. 在时间轴面板左侧工具栏中打开自动关键帧按钮。
2. 移动播放头到想要打关键帧的地方。
3. 选中一个子骨架，在属性面板中修改子骨架的播放动画。
4. 时间轴上会自动出现关键帧。
5. 鼠标拖动播放头，观察动画情况。

## 手动关键帧
1. 在时间轴面板左侧工具栏中关闭自动关键帧按钮。
2. 移动播放头到想要打关键帧的地方。
3. 选中一个子骨架，在属性面板中修改子骨架的播放动画。
4. 属性面板中，播放动画的属性后面的小旗子变成黄色，点击小旗子按钮，手动打关键帧。
5. 小旗子变成蓝色，表示当前播放头下有关键帧。
6. 鼠标拖动播放头，观察动画情况。
