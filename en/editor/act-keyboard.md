# Keyboard Shortcuts

Currently, there are only a few supported keyboard shortcuts, more will be added in the future

- Ctrl + S  Save
- Ctrl + C  Copy
- Ctrl + V  Paste
- Ctrl + X  Cut
- Ctrl + Z  Undo
- Ctrl + Y  Redo
- Number 1  Switch to skeleton mode
- Number 2  Switch to animation mode
- F12  Switch to fullscreen
